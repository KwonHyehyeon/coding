반복문

반복문은 프로그램에서 필요한 결과 값을 도출하기 위해 실행문의 순서를 반복적으로 실행시키는 문법을 말합니다. 빈복문에는 while문, do while문, for문이 있으며, for문을 가장 많이 사용합니다.

while문

for문은 특정 구문을 여러번 반복하고 싶을 때 사용하는 반복문입니다.

for ( 초기값; 조건식; 증감값 ){
    실행문
}
